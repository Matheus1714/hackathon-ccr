# Projeto Posto Certo



## Descrição

O PostoCerto é um app de avaliação de postos de combustível voltado para caminhoneiros. Nele, caminhoneiros podem visualizar diferentes postos próximos ao local onde ele está com base na nota que têm no aplicativo, auxiliando-os a escolher o melhor posto para sua parada. Após utilizar o estabelecimento, o motorista pode avaliar os serviços prestados, e, caso queira, escrever um comentário, elogio ou reclamação. A partir disso, os postos adquirem maior visibilidade frente a seus usuário mais significativos e descobrem quais aspectos de seus serviços necessitam de melhorias. O PostoCerto cria um canal de comunicação direto entre os caminheiros e os postos de combustível, favorecendo a melhoria dos serviços através de uma concorrência saudável entre os estabelecimentos. Com a melhoria destes serviços, os caminhoneiros melhoram sua segurança, sua saúde e seu bem estar, garantindo viagens mais proveitosas e reduzindo o risco de acidentes.
